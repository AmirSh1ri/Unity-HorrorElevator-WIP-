# Elevator Horror – Camera-Based Visibility Game

## Description
A first-person Unity horror game built around a **camera-based visibility mechanic**. Enemies are **not visible by default** and only appear when directly viewed through a **rechargeable in-game camera**, creating tension through limited visibility and resource management.

The project focuses on **camera systems, visibility checks, and controlled enemy reveal logic**, using player perspective as a core gameplay mechanic. The game is structured with future publishing and release in mind.

---

## Features
- Camera-based mechanic where enemies appear only when viewed through the camera  
- Rechargeable camera system with limited usage  
- 21 different rooms with different objectives
- Visibility and line-of-sight checks to control enemy reveals  
- Resource-based gameplay design to increase tension  
- First-person camera control and perspective-driven gameplay
